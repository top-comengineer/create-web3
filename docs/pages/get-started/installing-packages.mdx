# Installing Packages

When going to add packages, you will need to point to the correct workspace

### YARN

javascript install

```jsx
// to add packages to the Next Js Environment dependencies
yarn workspace @project/next-app add <package-name>


// to add packages to the hardhat Environment dependencies
yarn workspace @project/hardhat add <package-name>
```

typescript install

```jsx
// to add packages to the Next Js Environment dependencies
yarn workspace @project/next-app-ts add <package-name>


// to add packages to the hardhat Environment dependencies
yarn workspace @project/hardhat-ts add <package-name>
```
